Payroll Management System

I designed and developed a web-based payroll management system using PHP, HTML, and CSS. The purpose of the system was to streamline payroll processes for small businesses by automating tasks such as employee record management, salary calculation, and tax deduction. I created the front-end using HTML and CSS and used PHP to implement the back-end functionality, including database management and data processing. The system was designed to be user-friendly and scalable, allowing businesses to easily add or remove employees as needed. Through this project, I gained experience in web development, database management, and problem-solving.


Technologies used: HTML, CSS, PHP, MySQL
